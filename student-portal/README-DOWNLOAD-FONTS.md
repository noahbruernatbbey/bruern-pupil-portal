Download OpenDyslexic fonts into assets/fonts/

Run the bundled downloader script to fetch OpenDyslexic TTF files into `assets/fonts`.

Usage:

```bash
# from the project root
node download-fonts.js
```

If the script fails, manually download fonts from the official site:

- https://opendyslexic.org
- or the repository: https://github.com/antijingoist/open-dyslexic (check the `ttf/` or `ttf` directory)

Then place the font files into `assets/fonts/` and update your CSS if needed.
